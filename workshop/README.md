This workshop has been tested on the following platforms:

* Client:
  * **linux**: version 4.15.0
  * **OpenShift CLI**: version 4.5.0
  * **git**: version 2.17.1

  * **Docker CE**: version 19.03.6
  * **curl**: version 7.58.0
